# AI Scholar 启动说明

## 快速启动

1. **确保已安装 Node.js**
   - 下载地址: https://nodejs.org/
   - 推荐安装 LTS 版本

2. **首次启动前安装依赖**
   ```bash
   npm install
   ```

3. **启动应用**
   - 双击运行 `start.bat` 脚本
   - 或在命令行中执行:
     ```bash
     .\start.bat
     ```

4. **访问应用**
   - 打开浏览器访问: http://localhost:5173/
   - 如果端口被占用，Vite 会自动分配其他端口，请查看命令行输出

## 常见问题

### 问题 1: 双击脚本没有反应
- 确保脚本文件名为 `start.bat`，而不是 `start.bat.txt`
- 确保脚本位于项目根目录

### 问题 2: 提示 "npm 不是内部或外部命令"
- 请先安装 Node.js
- 安装完成后需要重启电脑或重新打开命令行窗口

### 问题 3: 依赖安装失败
- 检查网络连接
- 尝试使用国内镜像: `npm install --registry=https://registry.npmmirror.com`

### 问题 4: 应用无法访问
- 检查命令行输出，确认实际使用的端口
- 确保防火墙没有阻止 Node.js 访问网络

## 停止应用

- 在命令行窗口中按 `Ctrl + C` 停止应用
- 或直接关闭命令行窗口

## 开发模式

如果需要开发模式，可以直接在命令行中执行:
```bash
npm run dev
```

## 构建生产版本

```bash
npm run build
```

构建后的文件会生成在 `dist` 目录中。
